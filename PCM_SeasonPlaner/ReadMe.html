I. SUMMARY

The PCM Season Planner helps you to plan the races where your team is participating. The actual version allows you to define at which races your team should participate and which cyclists should ride the races.
PCM Season Planner is an editor for Cyanide databases (.cdb) for the game "Pro Cycling Manager - Tour de France 2016" but it might also work with PCM 2015 and PCM 2014.
This planner is an external program, therefore you need to save your game and quit the Cycling Manager before you can start using this tool.
The workflow ist quit simple. At the beginnning all the informations will be extracted from the saved game. Then you can plan your season and write the settings back to the save. Now you can continue playing the game. This can be done as often as you want and any time during the season.
The tool is available in English, German (and partially in French and Spanish) at the moment. It can be translated to any other language.

II. QUICK START

  1. Close the game or exit the career by going to the menu to save the career.
  2. Start the program PCM_SeasonPlanner.exe
  3. Select New plan from the start menu (the star), then select your save. (Extracting takes a while)
  4. Click Season planner in the menu bar.
  5. In the top section you can see all the races of the season.
  6. In the next section you can see the races your team has participated / will participate in.
  7. In the following sections you can see which riders participated / should participate.
    For all oncomming races you can asign riders by dragging the race bar to the rider.
  8. At the bottom of the screen, you set the fitness training schedule of the selected rider.
    Some helpers might not have an individual training schedule because they use the same schedule as their leader.
    You can leave it empty or copy a schedule from another rider which has a schedule.
  9. Select Export to cdb and enter a new name for your save

Attention: It is recommended to not overwrite your save.

IV. ADVANCED FUNCTIONS

You can change the language by clicking on Languages in the start menu (the star).
You can pause your planing and continue later. Please use Save plan and Open plan in the start menu to do so.
Attention: If you already continued your season in the game, then you have to use New plan to load the changed informations from your save.
You will be notified if you select too many cyclists for a race. If you do not like these hints, you can turn them off in the Options in the start menu.
The game will always fill missing cyclists to reach the maximum of allowed riders. If you do not like this you can use the hack. Activate it in the Options.

My Team - interface

Here you see all your cyclists. The layout is similar to the one in the game.
You can go to this interface by clicking My team in the menu bar at the top.
You can sort the list by clicking the column header. A small arrow shows the sorting direction. Click again to change the direction. Hold Shift to sort multiple columns.
Only in PCM 2016:The columns Objectives 1 to 3 can be used to change the objectives of the leaders. If you give objectives to rider who is not a leader he will become one. Attention: Assigning too many leader could crash the game.
The last two columns are used to arrange the cyclists on the Season planner. You can enter values manually or click the button above to use the sorting order you see at the moment.
You can see the number of the raced days and the planned race days on the tab Fitness training.

Season Plan - interface

Here you see all races of the season excluding national and world events. The timeline helps you to see which races overlap.
You can go to this interface by clicking Season planner in the menu bar at the top.
Use the filter above the first section to find all sponsor objectives. Hover your mouse over the race to see the sponsor goal.
In the second section you will find all races your team participates in. To add a race, just drag it from the top section. To remove a race, right click it. Please note that mandatory Pro-Tour races are handled by the game, ignoring your edits in my planner.
The cyclists are sorted as defined in the 'My team' interface. Adding/removing races works the same as for the team. If you are using cyclist groups, you can drag the race to the group header to assign it to all riders in the group.
You can see the number of the planned race days to the right of the name of each cyclist and the number of participating cyclists above.
You will be notified if you select too many cyclists. If you do not like these notes, you can turn them off in the the Options.
If you are a manager of a Conti-team it is possible to asign riders to Pro-Tour races, even if you do not know if you will receive the wild card.
Click on the name of the cyclists to select him. The section at the bottom shows the fitness training schedule of the selected cyclist including all the races he will be participating in.

V. INSTALLATION + UNINSTALLATION

Requirements

  • Microsoft XP (32/64), Windows Vista (32/64), Windows 7 (32/64), Windows 8 (32/64), Windows 10 (32/64)
  • Microsoft .Net Framework 3.5 SP1 *

* Windows 7, 8 and 10 have .Net 4.0 installed by default so you do not need .Net 3.5

Installation

Extract the archive into an empty folder on your drive. Do not delete any files or folders within.

Deinstallation

Delete the folder.

VII. STATUS + OPEN POINTS

This program can read saves from PCM2016 and maybe from PCM13, PCM14 and PCM15.
Please note that the planner does not support multiple plans at the moment. New plan will overwrite an existing plan.

What is not working

  • Due to a change in the database structure as of PCM2013 my planner cannot show the stage profiles but you can activate the race type icons.
  • Calculation of the fatigue based on the applied fitness schedule is not implemented. You have to check it in the game afterwards.
